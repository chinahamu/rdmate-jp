# DMPテンプレートの選び方

RDMate JPのDMPテンプレートは、研究分野、助成機関、利用言語、テンプレートバージョンを明示して管理します。このドキュメントでは、利用者が適切なテンプレートを選ぶための判断基準を説明します。

## 1. テンプレート選択の基本

最初に、次の順序で候補を絞り込みます。

1. 助成機関または学内ルールで指定されたDMP要件があるか確認します。
2. 分野固有のメタデータ標準やリポジトリ要件があるか確認します。
3. 日本語で学内共有するか、英語で国際共同研究者と共有するかを決めます。
4. 既存DMPから移行する場合は、互換バージョンを確認します。

指定要件がない場合は、汎用日本語テンプレートから開始します。

## 2. 既定テンプレート

| テンプレートID | 用途 | 推奨利用場面 |
| --- | --- | --- |
| `generic-jp-v2` | 汎用DMP日本語テンプレート | 学内研究、研究室単位のDMP、要件未確定の初期整理 |
| `japanese-funder-jp-v1` | JST/AMED/科研費向け日本語DMP | 日本の助成事業、課題番号や助成要件を明示したい場合 |
| `nih-style-v1` | NIH DMS Plan風テンプレート | 海外助成・国際共同研究の説明様式に寄せたい場合 |

## 3. 選択フロー

### 3.1 汎用研究・学内共有

- 助成機関指定がない。
- 研究室内や学内でDMPを共有する。
- 研究データの種類、保存場所、公開方針、責任者をまず整理したい。

この場合は `generic-jp-v2` を選択します。

### 3.2 日本の助成機関向け

- JST、AMED、科研費などの助成制度と紐づく。
- 助成機関名、制度名、課題番号、公開要件を明記したい。
- 申請時または採択後のDMP確認に使う。

この場合は `japanese-funder-jp-v1` を選択します。

### 3.3 国際共同研究・海外様式

- 英語圏の共同研究者と説明項目を合わせたい。
- NIH DMS Plan風の構成に近い整理が必要。
- データ共有、アクセス制限、保存期間の説明を海外様式に寄せたい。

この場合は `nih-style-v1` を選択します。

## 4. 回答時の注意点

- DMPには実データ本体を貼り付けないでください。
- 保存場所はRDMate JP内のファイルアップロードではなく、外部ストレージ、機関リポジトリ、GakuNin RDM、研究室サーバなどの参照情報として記載します。
- 個人情報、要配慮情報、共同研究契約、特許出願予定がある場合は、公開方針を即断せず、公開判断の根拠を残してください。
- Dataset台帳とDMPの記載がずれた場合は、`/dmp-diff` で差分を確認し、どちらを更新するか決めます。

## 5. テンプレート更新時の互換性

テンプレートには `id`、`version`、`compatibleTemplateVersions` を設定します。既存DMPを新しいテンプレートへ移行する場合は、次を確認します。

- 既存回答の質問IDが新テンプレートに残っているか。
- 必須質問が増えていないか。
- 選択肢の値が変わっていないか。
- 表示名だけの変更か、回答の意味が変わる変更か。

破壊的変更がある場合は、テンプレートIDまたはメジャーバージョンを変更してください。

## 6. 開発者向け補足

新しいテンプレートを追加する場合は `templates/dmp/` にJSONを追加します。作成手順とレビューポイントは `docs/template-authoring.md` を参照してください。
